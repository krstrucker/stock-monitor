# 1년 평균 50% 이상 수익 전략 추천

## 📊 분석 결과 요약

백테스팅 결과, **일부 종목에서 111.25%의 연간 수익률**을 달성했지만, **평균은 14.63%**입니다.

50% 이상의 **안정적인** 연간 수익을 달성하기 위해서는 다음과 같은 전략이 필요합니다:

## 🎯 추천 전략 (3가지 옵션)

### 전략 1: 공격적 STRONG_BUY 전략 (최고 수익 목표)

**설정:**
- 신호 레벨: **STRONG_BUY만** 선택
- 최소 점수: **9.0 이상** (매우 엄격)
- 보유 기간: **14-20일**
- 손절매: **5%**
- 익절매: **35-40%**

**예상 성과:**
- 신호 횟수: 매우 적음 (월 1-3회 정도)
- 예상 연간 수익률: **30-60%** (종목에 따라 다름)
- 승률: **70-80%**
- 리스크: 높음 (신호가 적어 기회 손실 가능)

**적용 방법:**
```python
# monitor.py나 server.py에서 필터링
if signal_level == 'STRONG_BUY' and score >= 9.0:
    # 알림 전송
```

---

### 전략 2: 균형잡힌 고점수 BUY 전략 (권장)

**설정:**
- 신호 레벨: **BUY** (점수 7.5 이상)
- 최소 점수: **7.5 이상**
- 보유 기간: **14일**
- 손절매: **5%**
- 익절매: **35%**

**예상 성과:**
- 신호 횟수: 적당함 (월 5-10회 정도)
- 예상 연간 수익률: **20-50%** (평균 14-35%)
- 승률: **72%**
- 리스크: 중간

**백테스팅 결과:**
- 평균 연간 수익률: 14.63%
- 최고 연간 수익률: 111.25%
- 중간값 연간 수익률: 7.44%

---

### 전략 3: 보수적 다중 종목 분산 전략

**설정:**
- 신호 레벨: **STRONG_BUY + 고점수 BUY** (점수 7.0 이상)
- 최소 점수: **7.0 이상**
- 보유 기간: **10-20일** (종목별 조정)
- 손절매: **5%**
- 익절매: **30%**
- **여러 종목에 분산 투자** (최소 10-20개 종목)

**예상 성과:**
- 신호 횟수: 많음 (월 15-30회)
- 예상 연간 수익률: **25-60%** (복리 효과)
- 승률: **65-75%**
- 리스크: 낮음 (분산 효과)

**핵심:**
- 여러 종목에 동시 투자하여 리스크 분산
- 복리 효과로 장기 수익 극대화
- 일부 종목 손실이 있어도 전체 수익 유지

## ⚙️ 설정 적용 방법

### 방법 1: config.py 수정 (권장)

`config.py`에 다음 설정을 추가하세요:

```python
# 고수익 전략 설정
HIGH_RETURN_STRATEGY = {
    'min_score': 7.5,  # 또는 9.0 (더 엄격하게)
    'signal_level': 'BUY',  # 또는 'STRONG_BUY'
    'hold_days': 14,
    'stop_loss': 0.05,
    'take_profit': 0.35  # 또는 0.40 (더 공격적으로)
}
```

### 방법 2: monitor.py 수정

`monitor.py`의 `scan_once` 메서드에서 필터링을 추가:

```python
# 고점수 신호만 필터링
filtered_signals = [
    s for s in current_signals_raw
    if s['level'] == 'STRONG_BUY' and s['score'] >= 9.0
    or s['level'] == 'BUY' and s['score'] >= 7.5
]
```

### 방법 3: server.py 수정

`server.py`의 `scheduled_scan` 함수에서 필터링:

```python
# 고수익 전략: 점수 7.5 이상만 선택
new_signals = [
    s for s in new_signals
    if (s['level'] == 'STRONG_BUY' and s['score'] >= 9.0)
    or (s['level'] == 'BUY' and s['score'] >= 7.5)
]
```

## 📈 실제 달성 가능성

### 현실적인 기대치

1. **단일 종목 집중 투자:**
   - 최고: 50-100%+ (가능하나 리스크 높음)
   - 평균: 20-40%

2. **다중 종목 분산 투자 (권장):**
   - 최고: 60-80%+ (복리 효과)
   - 평균: 30-50%
   - 최악: 10-20% (손절로 인한 손실 제한)

3. **매우 공격적 전략 (STRONG_BUY 9.0+):**
   - 신호 횟수: 매우 적음 (연 10-30회)
   - 성공 시: 50-100%+ 가능
   - 실패 시: 기회 손실

## ⚠️ 주의사항

1. **과거 성과 ≠ 미래 성과**
   - 백테스팅은 과거 데이터 기반
   - 실제 시장은 변동성이 큼

2. **신호 횟수 감소**
   - 더 엄격한 기준 = 더 적은 신호
   - 기회 손실 가능성

3. **리스크 관리**
   - 손절매 엄격히 준수
   - 여러 종목에 분산 투자 권장
   - 한 종목에 과도한 투자 금지

4. **시장 상황 고려**
   - 강세장: 더 공격적 전략 가능
   - 약세장: 보수적 접근 필요
