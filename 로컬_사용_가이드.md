# 로컬 컴퓨터에서 사용하기

## 간단한 방법

로컬 컴퓨터에서 사용하는 것이 더 간단할 수 있습니다!

### 장점
- ✅ 추가 설치 불필요
- ✅ 설정이 간단
- ✅ 무료
- ✅ 즉시 사용 가능

### 단점
- ⚠️ 컴퓨터를 켜놔야 함
- ⚠️ 전력 소비

## 사용 방법

### 방법 1: 배치 파일 사용 (가장 간단)

1. `start_server_background.bat` 파일 더블클릭
2. 새 창이 열리면서 서버 실행
3. 컴퓨터를 켜놓기

### 방법 2: 작업 스케줄러로 자동 시작

컴퓨터를 켤 때마다 자동으로 서버가 시작되도록 설정:

1. **작업 스케줄러 열기**
   - Win + R → `taskschd.msc` 입력

2. **기본 작업 만들기**
   - 오른쪽 "기본 작업 만들기" 클릭
   - 이름: "주식 신호 모니터링 서버"

3. **트리거 설정**
   - "컴퓨터 시작 시" 선택

4. **동작 설정**
   - "프로그램 시작" 선택
   - 프로그램/스크립트: `start_server_background.bat`의 전체 경로
     - 예: `C:\Users\용\Desktop\us stock auto\start_server_background.bat`
   - 시작 위치: `C:\Users\용\Desktop\us stock auto`

5. **완료**
   - 이제 컴퓨터를 켤 때마다 자동으로 서버가 시작됩니다!

## 절전 모드 방지

서버가 계속 작동하려면 절전 모드를 비활성화하세요:

1. **제어판** → **전원 옵션**
2. **절전 모드 설정 변경**
3. **컴퓨터를 절전 모드로 전환** → **안 함** 선택

## 추천 설정

- 컴퓨터를 켜놓기
- 절전 모드 비활성화
- 작업 스케줄러로 자동 시작 설정

이렇게 하면 컴퓨터를 켤 때마다 자동으로 서버가 시작됩니다!
