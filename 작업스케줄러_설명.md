# Windows 작업 스케줄러 설명

## ⚠️ 중요: 작업 스케줄러의 한계

**Windows 작업 스케줄러는:**
- ✅ 컴퓨터가 **켜져 있을 때** 자동으로 프로그램을 실행합니다
- ❌ 컴퓨터가 **꺼져 있으면** 아무것도 실행할 수 없습니다
- ❌ 컴퓨터를 **자동으로 켜는 기능은 없습니다**

## 새벽에 스캔이 실행되지 않은 이유

1. **컴퓨터가 꺼져 있었을 가능성**
   - 컴퓨터가 꺼져 있으면 작업 스케줄러도 작동하지 않습니다
   - 서버도 실행되지 않습니다

2. **서버가 실행되지 않았을 가능성**
   - 파일이 없어서 서버를 실행할 수 없었습니다 (이제 해결됨)
   - 수동으로 서버를 실행하지 않았을 수 있습니다

## 해결 방법

### 방법 1: 컴퓨터를 항상 켜두기 (가장 간단)
- 컴퓨터를 절전 모드나 최대 절전 모드로 설정
- 또는 컴퓨터를 완전히 끄지 않기
- 작업 스케줄러가 자동으로 서버를 실행

### 방법 2: Wake-on-LAN 설정 (고급)
- 컴퓨터를 원격으로 켤 수 있도록 설정
- 네트워크를 통해 컴퓨터를 깨우기
- 복잡하고 하드웨어/네트워크 설정이 필요

### 방법 3: 클라우드 서버 사용
- Heroku, AWS, Google Cloud 등 클라우드 서버 사용
- 24시간 항상 켜져 있음
- 비용 발생

## 작업 스케줄러 등록의 목적

작업 스케줄러를 등록하면:
- ✅ 컴퓨터가 **켜져 있을 때** 자동으로 서버 실행
- ✅ 컴퓨터 재시작 후 자동으로 서버 실행
- ✅ 로그인하지 않아도 서버 실행
- ❌ 컴퓨터가 꺼져 있으면 실행 불가

## 권장 사항

**가장 실용적인 방법:**
1. 컴퓨터를 절전 모드로 설정 (전력 소모 최소화)
2. 작업 스케줄러에 등록하여 자동 실행
3. 컴퓨터가 켜져 있는 동안만 스캔 실행

**또는:**
- 컴퓨터를 완전히 끄지 않고 절전 모드로만 사용
- 작업 스케줄러가 스케줄 시간에 컴퓨터를 깨울 수 있도록 설정

## 결론

**작업 스케줄러는:**
- 컴퓨터가 **켜져 있을 때** 자동 실행
- 컴퓨터가 **꺼져 있으면** 실행 불가
- 컴퓨터를 **자동으로 켜는 기능은 없음**

**새벽 스캔을 위해서는:**
- 컴퓨터를 항상 켜두거나 절전 모드로 설정
- 또는 클라우드 서버 사용
